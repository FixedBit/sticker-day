# D20 Thermal Sticker Generator

A browser-based tool for creating custom stickers optimized for the D20 Mini Sticker Printer and other 57mm thermal printers.

## Live Demo

Visit the generator: [https://yourusername.github.io/sticker-day/](https://yourusername.github.io/sticker-day/)

## Quick Start

1. Open `index.html` in your browser
2. Choose a sticker type from the dropdown
3. Customize your content and dimensions
4. Export and print via your D20 printer app

## Features

- **Multiple Templates**: Habit trackers, to-do lists, date stamps, and more
- **Pure Black & White**: Optimized for thermal printing
- **57mm Width**: Perfect fit for standard thermal paper rolls
- **Customizable**: Adjust height, add items, change styles
- **Batch Export**: Generate multiple copies at once

## About the D20 Printer

The D20 Mini Sticker Printer is a pocket-sized thermal printer that uses:
- **Technology**: Inkless thermal printing (monochrome)
- **Connectivity**: Bluetooth
- **Paper**: 57mm wide thermal/sticker rolls
- **Apps**: Nada Print, Jaden's printer app, and others
